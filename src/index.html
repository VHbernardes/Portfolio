Projetos

Projeto 1

forUnB é um fórum inspirado no Stack Overflow, criado para fomentar a colaboração e o compartilhamento de conhecimento entre estudantes da Universidade de Brasília. Desenvolvido em Django com Bootstrap, usa PostgreSQL para o banco de dados e é hospedado no Render. Além disso, integra web scraping com BeautifulSoup para coletar materiais acadêmicos da UnB.

Projeto 2

O Gerenciador de Tarefas é uma aplicação web baseada no modelo Kanban, desenvolvida em Java com Spring Boot e Thymeleaf. Ela permite criar, atualizar e excluir tarefas de forma intuitiva, organizando-as nos status A Fazer, Em Progresso e Finalizado. Os dados são armazenados em PostgreSQL, e a aplicação é conteinerizada com Docker, para facilitar a hospedagem no Render.

Projeto 3

O projeto de Requisitos de Software teve como foco a análise e especificação do Sympla, uma plataforma de venda de ingressos e gestão de eventos. Durante o desenvolvimento, aplicamos conceitos essenciais da engenharia de requisitos, incluindo levantamento de requisitos, modelagem de casos de uso e definição de personas, garantindo uma visão clara das necessidades dos usuários e do sistema. Além disso, o projeto teve um olhar crítico sobre requisitos não implementados na plataforma, identificando melhorias e oportunidades para evolução do software. O trabalho envolveu desde a análise do mercado e stakeholders até a documentação detalhada dos requisitos funcionais e não funcionais.
